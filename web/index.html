The source and packages you can find here add the following enhancements to the upstream version:
  • Better Mac OS X support
  • Better Microsoft Windows support (standalone installer provided here)
  • JPEG image compression
  • Adaptive JPEG mode (bandwidth constrained)
  • Support for password protection option (useful for securing TCP mode)
  • Ability to disable pulseaudio and clipboard synchronization
  • Support for using existing displays or Xdummy in place of Xvfb, RANDR support
  • Lots of small fixes
